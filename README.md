Siina
=====
 **Siina**는 **다중 오실레이터** 및 **마스터 볼륨(앰프) ADSR**이 지원되는 간단한 웹 기반 피아노 앱입니다.

 군 복무 기간 중에 심심풀이로 개발을 시작하여 약 1년 동안 업데이트를 했던 토이 프로젝트입니다.

  - 이전 버전 **v0.0.4-preview** : 브랜치 *(업로드 예정)*, 릴리즈 *(업로드 예정)*
