
=====

 **** 환경에서도 (유사) 피아노를 연주할 수 있는 웹 앱, Siina를 사용해주셔서 감사합니다!

 이 웹 앱은 **** 환경 특성 상 서버에 배포되는 것이 아닌, 사용자가 직접 다운로드받아 웹 브라우저로 직접 실행하는 방식을 선택하고 있습니다.

 이 퀵스타트 설명 파일에는 Siina를 실행하는 방법에 대해 기술되어 있으며, 사용 방법은 Siina 앱 내부에서 확인하실 수 있습니다(아직 미구현입니다).

=====

 [ 시스템 요구 사항 ]

 이 웹 앱은 기본적으로 고성능의 컴퓨터를 필요로 하지 않습니다.

  1. (필수) 웹 표준을 준수하는 HTML5 지원 최신 웹 브라우저
    └ 공군 IT게시판에 배포되는 최신 Chrome (v90 이상)
	  └ 배포일 기준 최신 Chrome 다운로드 게시글 : http://***.**.***/*****
	└ Windows 10 버전 1909 이상에서 자동 설치되는 내장 Microsoft Edge (Chromium 기반의 새로운 Microsoft Edge)
	    └ 이전 Microsoft Edge(EdgeHTML 기반)는 지원되지 않습니다.

  2. (필수) 마우스 등 포인팅 장치
  
  3. (추천) 키보드

=====

 [ 실행 방법 ]

  1. 배포지에서 Siina를 다운로드 받습니다.
    └ 주로 "*** MSM-Wiki"에 배포하고 있습니다.
	  └ http://***.**.***/*****
	  └ **** 로그인 및 *** 카페 가입이 필요할 수 있습니다.
	└ 압축 파일의 형태로 배포하고 있으며, 압축 해제가 필요합니다.
	└ "원 제작자 전역 후" 다른 전우님께서 수정 혹은 재배포하실 때(감사합니다!!) 배포지 및 압축 형식이 달라질 수 있습니다.

  2. 적당한 곳에 다운로드 받은 압축 파일을 해제합니다.
    └ 경우에 따라 반디집 등 외부 프로그램이 필요할 수 있습니다.
	  └ 반디집 또한 IT게시판에서 다운로드 받으실 수 있습니다.
	    └ http://***.**.***/*****

  3. 압축 해제한 폴더 내의 "siina.html"을 웹 브라우저로 실행합니다.
    └ 파일에 마우스 우클릭 → 연결 프로그램(H) 메뉴에서 웹 브라우저를 선택하거나,
	   실행 중인 웹 브라우저의 주소 표시줄에 파일을 드래그 앤 드롭하여 실행 가능합니다.

 [ 주의 사항 ]
 
  ※ 명시된 웹 브라우저 버전 이전의 버전에선 정상 작동을 보장하지 않습니다. 이 경우 기능상 문제가 발생하여도 응대하지 않습니다.
     가능한 한 최신 버전의 웹 브라우저를 사용해주시기 바랍니다.

  ※ 자신의 웹 브라우저 버전은 웹 브라우저 프로그램 내 "정보" 페이지 또는 Siina의 정보 플라이아웃에서 확인 가능합니다.
  
  ※ 이 웹 앱은 앱 설정 저장 목적으로 웹 브라우저의 LocalStorage 기능을 활용합니다.

  ※ 이 웹 앱은 앱 폴더 내의 로컬 파일에만 접근합니다. 앱 폴더를 벗어나는 외부 파일 및 인트라넷 네트워크에 접근하지 않습니다.
  
  ※ 이 웹 앱은 사용자 및 PC의 민감 정보를 수집하지 않습니다.

  ※ 이 웹 앱을 전송, 사용 및 저장 장치에 보관함으로서 발생하는 불이익은 원 제작자 및 수정·배포자에게 책임이 없고, 모두 사용자에게 귀속됩니다.
     Siina를 다운로드받는 순간부터 이에 동의한 것으로 간주합니다.
